# spring-cartao-credito-sample-api
About Controle das Movimentações do Cartão de Crédito de uma Família (CRUD Básico) com API Spring Boot 4.0.2 no Backend e Angular 21 no Front (CLI 21.1.4 e Node.js 24.11.1) usando Standalone Components (angular-cartao-credito-sample-app). A API usa o BD H2 Database Engine.
